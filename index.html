The ultimate shopping companion. Works in the background on all supported stores.

📱 Installing on Mobile? View Guide
How to install on Mobile:
  1. Tap Copy Code below to copy the bookmarklet script.
  2. Bookmark this page (tap your browser's share or menu icon, and tap Add Bookmark).
  3. Edit the bookmark: Change its name to ⚡ STYLA Stylist, and paste the copied code into the Address/URL field. Save it.
  4. To use: Open any online store, type ⚡ STYLA Stylist in your browser's address bar, and tap the bookmark to launch the Stylist popup.
