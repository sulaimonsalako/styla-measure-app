💡 Sizing & Tech Pack Guide (Click to Expand)

Creating a factory-ready size chart requires translating your base sample garment dimensions into a full grading spec sheet. Styla's AI assistant automates this by applying garment industry grading rules.

1. Point of Measure (POM)

Specify critical dimensions. Standard POMs include Chest Width (measured 1" below the armhole), Body Length (from High Point Shoulder to hem), and Sleeve Length.

2. Fabric & Silhouette Controls

Fit Silhouette defines standard ease boundaries (oversized, regular, slim). Fabric Stretch Index determines sizing tolerances (woven fabrics require higher grade jumps than stretch knitwear).

3. Shopify / Web Integration

Once generated, click "Copy Shopify Size Table". In your Shopify store admin, open a product description, switch to HTML mode, and paste the code for a beautifully styled size table.
